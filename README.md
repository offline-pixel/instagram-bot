#Instagram Bot
###prerequisite 
1. You must have installed node.js latest version. Otherwise, Go to [node.js](https://nodejs.org/en/)
2. You know how to run puppeteer in the editor

###Steps
1. cd to root folder, open index.js
2. Enter `username` and `password`
3. Specify yuor tags
3. Run `npm install`
4. Launch puppeteer :)
